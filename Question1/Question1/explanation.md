## Question 1 – Command Explanations

1. whoami – Displays the name of the currently logged-in user.
2. groups – Shows all groups associated with the current user.
3. pwd – Prints the current working directory path.
4. ls -l – Lists files and folders in long format with permissions.
5. echo "Linux user environment verified" > user_info.txt – Creates a text file and writes the given content into it.
6. wc -m user_info.txt – Shows the total number of characters present in the file.
7. man mkdir – Opens the manual page for the mkdir command. Useful option: `-p` creates nested directories without error.
8. ls ~ | sort – Lists home directory contents in alphabetical order.
9. grep "admin" log.txt – Searches for the word 'admin' inside log.txt.
10. uname -r – Displays kernel version of Linux.
11. ping -c 4 8.8.8.8 – Tests network connectivity using ICMP packets (Internet working proof).
12. uptime – Shows how long the system has been running, number of users, and load average.
